Distributed Library Project

## Licence

## Install and Open Library

```
$ docker-compose build
$ docker-compose run api python migrate.py
```

to install dependent scripts...

after that, 

```
$ docker-compose up
```

to open Library,

and access to [localhost:8000](localhost:8000).
